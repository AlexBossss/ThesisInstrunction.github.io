Сценарий работы инициатора
Инициатор создает карточку, заполняет необходимые ему поля, добавляет вложения и запускает процесс согласования, выбирая участников процесса. В случае отправки карточки на доработку от инициатора требуется внести изменения (изменить поле, вложение, участников процесса).
Как создать карточку документа
Создать можно как новую карточку с пустыми полями, так и карточку с уже заполненными полями используя шаблоны документов или скопировать существующую карточку.
Создать новую карточку:
  • на главном экране системы
  • пункт меню «Документы» – «Создать документ»
  • из списка документов «Создать» - «Новый»
  • из списка шаблонов «Создать документ»
  • из карточки документа «Создать» - «На основании»
  • используя плагины
Как заполнить карточку
Основные поля для заполнения находятся на вкладке детали. Обязательные поля помечены символом *. Для быстрого заполнения полей можно воспользоваться шаблонами, нажав соответствующую кнопку в правой части экрана.
Как добавить вложение
Перейдите на вкладку Вложения и выберите один из способов:
  • добавить файл (или файлы) с жесткого диска
  • добавить файл из буфера
  • добавить файл со Сканера
  • добавить файл из подчиненых карточек
  • добавить версию вложения
Если вы являетесь создателем вложения, вы можете редактировать и изменять его на любых этапах. Удалять вложения других пользователей нельзя.
Как запустить процесс согласования
  1. В правой части экрана нажмите «Согаосование»
  2. Выберите пользователей на роли Согласующий, Утверждающий, Секретарь
  3. Для роли согласующий возможно выбрать несколько пользователей, использовав множественное назначение
  4. При необходимости установите длительности на этапы и оставьте комментарий
  5. Запустите процесс нажав «ОК»
Что такое доработка
В случае если участником процесса карточка было отклонена, она возвращается инициатору на доработку. Найти ее можно в папке документы — доработка. Инициатор вносит изменения в карточку и продолжает процесс. Время на доработку ограниченно, если при запуске процесса у инициатора стояла длительность.
Как изменить участников процесса
После запуска процесса Инициатор может поменять участников, которые не завершили действие (если включена функция Изменения участников).
  1. В правой части экрана нажмите «Изменить участников»
  2. Выберите новых пользователей на роль(и)
  3. Добавьте комментарий
  4. Нажмите «Ок»
